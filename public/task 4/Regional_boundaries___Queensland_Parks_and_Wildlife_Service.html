Data Quality

Data quality scope:
Level: dataset
Completeness (omission):

All known management changes have been incorporated.
Consistency (conceptual):

QPWS Regions consists of polygons in a single layer. Polygons are based on the DCDB dataset where possible, with other sources used as necessary. Any errors found are corrected and the data reprocessed.
Positional accuracy (external, absolute):

The dataset is being aligned to the spatial accuracy of the DCDB which will vary across Queensland.
Attribute accuracy (non quantitative):

This layer contains the Regions as defined by the QPWS administrative hierarchy.

History:

Lineage statement:
The previous version was based on a variety of spatial sources. Using topology edit tools, the Region layer is being aligned to the the DCDB parcels. The layer is also amended to reflect changes in the operational management of the estate.
